21 // This class encapsulates the (simplified) overlay algorithm.
22 // The algorithm proceeds in four steps.
23 // Step 1: Project blue vertices onto G.
24 // Step 2: Compute intersection of blue edges with green
25 // edges, and insert intersections into blue edges.
26 // Step 3: Sort intersections on green edges.
27 // Step 4: Project green vertices on B.
