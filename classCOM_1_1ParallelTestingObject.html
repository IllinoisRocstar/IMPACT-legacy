Detailed Description

template<typename CommType, typename ResultsType>
class COM::ParallelTestingObject< CommType, ResultsType >

The project-specific parallel testing object encapsulates all parallel tests for the project. This is the interface to all "in-core" parallel testing available for the given project. It implements the interface given by COM::TestingObject and has an IRAD::Comm::CommunicatorObject for parallel capabilities.
