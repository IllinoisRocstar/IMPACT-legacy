static void COM::TestModule::Load ( const std::string &  name)
inlinestatic

This function creates an instance of a COM::TestModule object, creates a new Window with the specified name and registers the address of the new COM::TestModule object as the ".global" DataItem of the Window. This registers the C++ COM::TestModule "object" with the COM runtime system.

Next, several member functions are registered. The first argument to each member function is a COM_RAWDATA type which will be the address of the object registerd to the ".global" DataItem.

Parameters
nameconst std::string specifying the name of the window to load the module into.
Note
This function must be a static function in order for it to work in the COM_LOAD_MODULE interface.
The COM_VOID type has been used here to pass in a pointer to a std::string object. This is cool.
