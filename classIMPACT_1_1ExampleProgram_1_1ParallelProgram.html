Detailed Description

This object encapsulates the example parallel program. It inherits from the IRAD::Global::Program type.

The program itself calculates by integrating over the interval [0,1]:

This method is MPI parallel and uses domain decomposition to calculate the total integral. On N processors, the n partitions of the domain, [0,1], are partitioned as follows:

With and , each processor performs the integration on its own portion of the domain:

The approximate values for the integrals on each processor, , are determined by numerical quadrature using two different methods:

  1. IMPACT::TrapezoidQuadrature
  2. IMPACT::MidPointQuadrature

The results of these quadrature methods from each processors are then summed to recover a total numerical estimate for :

The approximate value, and the error, are reported to stdout or, if specified, to a file.

Note
If the number of partitions, n, is less than the number of processors, N, then the program behaves as if it were serial, performing all calculations on processor 0.

The usage for this program goes like:

                Mode-dependent input: Input <filename> for serial example
                program, or <number of divisions> for parallel example.
