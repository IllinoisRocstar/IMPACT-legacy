Detailed Description

template<typename CommType, typename ResultsType>
class IMPACT::ParallelTestingObject< CommType, ResultsType >

The project-specific parallel testing object encapsulates all parallel tests for the project. This is the interface to all "in-core" parallel testing available for the given project. It implements the interface given by IMPACT::TestingObject and has an IRAD::Comm::CommunicatorObject for parallel capabilities.

This function implements a simple parallel test of the function IMPACT::MidPointQuadrature - which is designed to integrate a function, f, over an n-way partitioned interval in parallel.

The test first makes sure that the method integrates a linear function exactly, and then that the error term has the proper dependence on the spacings. Then the parallel scaling of the algorithm is checked by running on subsets of the total number of processors and checking time-to-solution.
